# 🏥 Cabinet Médical - Application de Gestion

Bienvenue dans l'application **Medicare Pro System**. C'est une solution complète pour la gestion administrative et médicale d'un cabinet.

## 🚀 Fonctionnalités
* **Tableau de bord :** Statistiques et prochains rendez-vous.
* **Gestion des Patients :** Création, modification et historique médical.
* **Rendez-vous :** Planning interactif avec tri et statuts.
* **Sécurité :** Espaces séparés pour Admin, Médecins et Secrétaires.
* **Design :** Interface moderne et responsive (Bootstrap 5).

## 🛠️ Technologies utilisées
* **Backend :** C# / ASP.NET MVC 5
* **Données :** Entity Framework 6 / SQL Server
* **Frontend :** HTML5, CSS3, Bootstrap 5, JavaScript
* **Outils :** Visual Studio 2022, Git

## ⚙️ Comment installer le projet

1.  **Cloner le projet :**
    ```bash
    git clone [https://github.com/TON-NOM/CabinetMedical.git](https://github.com/TON-NOM/CabinetMedical.git)
    ```
2.  **Base de données :**
    * Ouvrez SQL Server ou Visual Studio.
    * Exécutez le script `BaseDeDonnees_Maroc.sql` fourni dans le projet.
    * Cela créera la base `CabinetMedicalDB` avec des données de test (Patients et Médecins marocains).
3.  **Lancer l'application :**
    * Ouvrez la solution dans Visual Studio.
    * Appuyez sur `F5` (Démarrer).
